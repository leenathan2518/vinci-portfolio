# Junchen Li Portfolio Website

This repository contains my personal portfolio website, built as a lightweight static webpage and deployed through Cloudflare Workers.

The website serves as a central hub for my data analytics, applied economics, financial research, and automation projects. It links to deployed dashboards, GitHub repositories, research writing, and contact information.

## Website Purpose

The purpose of this portfolio website is to present selected projects that combine:

- Applied economics and market analysis
- Data analytics and visualization
- Python-based automation
- SQL and database workflows
- Interactive dashboard development
- Financial and policy research

Rather than acting as a standalone application, this website functions as a professional entry point to my broader project portfolio.

## Featured Projects

### Legal Case Analytics Dashboard

An interactive dashboard for analysing legal case management data using Python, MySQL, Streamlit, Pandas, and Plotly.  
The project demonstrates SQL-based data cleaning, privacy-preserving anonymisation, dashboard design, and operational analytics.

### US Macroeconomic Indicators Dashboard

A Streamlit dashboard for exploring key US macroeconomic indicators, including inflation, unemployment, GDP, interest rates, and recession-period patterns.  
The project focuses on time-series analysis, economic interpretation, and interactive visualisation.

### China A-Share Market Dashboard

An interactive dashboard for analysing China's A-share market through market overview, real-time market snapshots, sector rotation, capital-flow tracking, semiconductor ecosystem analysis, and risk-return comparison.  
The project uses Python, Streamlit, Pandas, Plotly, AKShare, and BaoStock.

### Automated Data Pipeline

A data engineering project that extracts, transforms, validates, and loads market and macroeconomic data into PostgreSQL.  
The project demonstrates ETL design, data quality checks, database workflows, modular Python scripts, logging, and dashboard-based monitoring.

### Equity Research Platform

A multi-company equity research dashboard focused on China’s new energy vehicle sector.  
It combines technical indicators, financial metrics, business performance, cost pressure, news signals, and a scoring framework into an interactive research platform.

### US Macro Shock Transmission Dashboard

An event-study dashboard analysing how US macroeconomic surprises affect global equity indices, bond yields, foreign exchange, commodities, and cross-market asset reactions.  
The project combines macro event classification, event-window analysis, cross-asset comparison, and financial market interpretation.
